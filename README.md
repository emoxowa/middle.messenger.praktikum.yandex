
# Messenger

Обучающий проект в рамках курса «Мидл фронтенд-разработчик» Яндекс.Практикум.

## Установка и запуск проекта

Установка зависимостей

    npm i

Запуск в режиме разработки

    npm run dev

Сборка проекта для продакшена

    npm run build

Запуск сервера

    npm run start
